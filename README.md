# 📊 **Análisis Exploratorio de Datos (EDA) – Clientes**

## 📌 **Descripción**

Este proyecto consiste en un análisis exploratorio de datos (EDA) utilizando Python, con el objetivo de identificar patrones en el comportamiento de clientes a partir de variables como ingresos, gasto mensual y frecuencia de compra.

## 🛠️ **Herramientas utilizadas**

* Python
* Pandas
* Matplotlib

## 📊 **Análisis realizado**

Se llevó a cabo un proceso de exploración y análisis de datos que incluyó:

* Revisión de estructura y tipos de datos
* Identificación de valores nulos
* Análisis estadístico descriptivo
* Evaluación de ingresos promedio por ciudad
* Visualización de la relación entre ingresos y gasto mensual
* Análisis de la distribución de ingresos

Además, se creó una métrica adicional (ratio de gasto) para evaluar la proporción del gasto respecto a los ingresos de cada cliente.

## 📈 **Resultados**

Se identificó una relación positiva entre ingresos y gasto mensual, lo que indica que clientes con mayores ingresos tienden a gastar más. También se evidenciaron diferencias en el comportamiento de los clientes según su ciudad.

El análisis del ratio de gasto permitió detectar clientes con alto nivel de consumo en relación a sus ingresos, lo cual puede ser útil para segmentación o evaluación de riesgo.
